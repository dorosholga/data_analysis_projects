# Создание дашборда по пользовательским событиям для агрегатора новостей

## Ссылка на дашборд

https://public.tableau.com/views/Dashboard_16563654964170/sheet4?:language=en-US&:display_count=n&:origin=viz_share_link   



## Описание

Установлен PostgreSQL, развернута база данных. Написан скрипт пайплайна, который собирает данные за определенный временной период, и настроена его автономная работа через crontab. Для визуализации данных  написан скрипт дашборда с несколькими фильтрами. По результатам была подготовлена презентация с графиками.


## Инструменты

- Python
- SQLAlchemy
- PostgreSQL
- dash
- Tableau
